# 介绍

北信科校园博客由北京信息科技大学(北信科，BISTU)的在校学生自发创建，目的是为志愿报考的新生和在校学生提供有限的信息资料，帮助同学更好的融入校园生活。

*注意：本博客非学校官网网站，站内内容请自行甄别。*

# 网站搭建

* 博客框架：[Hugo](https://gohugo.io/)
* 博客主题：[LoveIt](https://github.com/dillonzq/LoveIt)
* 评论系统：[Giscus](https://giscus.app/zh-CN) (基于 GitHub Discussions)
* 部署平台：[GitHub Pages](https://pages.github.com/)
* 加速服务：[Cloudflare](https://www.cloudflare.com/)
* 域名：[bistublog.xyz](https://bistublog.xyz)
* 地图服务：[高德开放平台](https://lbs.amap.com/)
* AI: Gemini
  
# 仓库结构

本仓库分为两个分支

* `main` 分支：存放 Hugo 博客的完整源码、配置文件及 Markdown 文章。
* `gh-pages` 分支：仅存放 `hugo` 编译生成的纯静态 HTML 网页文件。
